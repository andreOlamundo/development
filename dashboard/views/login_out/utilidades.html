Manual Tel. Panasonic KX-DT321

PANASONIC KX-DT321

  • Iniciar uma ligação

    9 + número ou SP-PHONE + número

  • Conferência interna

    número do ramal + tecla conferência + número do ramal

  • Conferência externa

    9 + número + tecla conferência + 9 + número + conferência (até oito pessoas)

  • Captura de qualquer ligação

    55

  • Transferência

    Tecla Transfer + número

  • Habilitar Caixa de Correio

    *381 + Gravar Mensagem + Tecla AutoDial

  • Chamar Portaria

    # ou 50

  • Ouvir mensagens da Caixa Postal

    Retirar fone do gancho + tecla MESSAGE
    Após ouvir Tecle 3 + AUTODIAL

  • Habilitar transferência interna (entre ramais)

    *7102 + número do ramal + tecla #

  • Desabilitar transferência interna (entre ramais)

    *7100

  • Habilitar transferência externa

    *7102 + 9 + número externo + tecla #

  • Desabilitar transferência externa

    *7100

  • Cadastro na lista pessoal

    Tecla PROGRAM + AUTODIAL + Tecla Navegacao (seta para baixo) + 9 + número + AUTODIAL + cadastre o nome + AUTODIAL

  • Iniciar uma ligação pela lista pessoal

    Tecla Navegacao (seta para direita) + Lista pessoal( display) + Tecla Navegacao (seta para cima) + tecle SP-PHONE

  • Excluir cadastro na lista pessoal

    Tecla PROGRAM + AUTODIAL + Tecla Navegacao (seta para baixo) + selecione o número (teclar duas vezes) TRANSFER + AUTODIAL
